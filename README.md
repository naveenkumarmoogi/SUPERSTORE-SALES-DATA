Week 1 — Microsoft Excel Dashboard Project
Overview

This repository contains my Week 1 project from my  Data Analytics Challenge.

The goal was simple:
Start Excel from scratch and build a complete business dashboard by the end of the week.

After 7 days of consistent practice, I created an end-to-end interactive sales dashboard using only Microsoft Excel.
